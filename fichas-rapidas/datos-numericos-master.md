## 1. TEMPERATURAS (oC)

| Lugar / Equipo | Temperatura |
|---|---|
| **Habitacion del paciente** | **20-22oC** |
| **Pasillos** | **20-21oC** |
| **Consultas** | **21-23oC** |
| **Quirofano** | **20-24oC** |
| **Incubadora neonatal** | **36-37oC** |
| **Deposito mortuorio / tanatorio** | **2-4oC** |
| **Autoclave (programa 1)** | **121oC durante 15-20 minutos** |
| **Autoclave (programa 2 - flash)** | **134oC durante 3-7 minutos** |
| **Poupinel / estufa calor seco (programa 1)** | **170oC durante 1 hora (60 min)** |
| **Poupinel / estufa calor seco (programa 2)** | **180oC durante 30 minutos** |

### Nemotecnicos temperaturas:
- "Habitacion = 20-22 (como en casa)"
- "Mortuorio = 2-4 (como el frigorifico)"
- "Quirofano = 20-24 (rango mas amplio)"
- "Autoclave: 121/15 y 134/3-7 (mas temp = menos tiempo)"
- "Poupinel: 170/60 y 180/30 (siempre mucho MAS que autoclave)"

## 6. TIEMPOS

### Tiempos clinicos y asistenciales

| Concepto | Tiempo |
|---|---|
| **Cambio de mascarilla quirurgica** | Cada **2 horas** (o si se humedece) |
| **Masaje en zona de presion (antiescaras)** | **4 minutos** por zona |
| **Cambios posturales prevencion UPP** | Cada **2-4 horas** |
| **Fallecido en planta: tiempo minimo en habitacion** | **2 horas** (minimo antes de trasladar a mortuorio) |

### Rigor mortis

| Fase | Tiempo |
|---|---|
| **Inicio del rigor mortis** | **2-4 horas** post-mortem |
| **Rigidez maxima** | **12-15 horas** |
| **Desaparicion del rigor** | **24-36 horas** |

### Tiempos de enterramiento

| Concepto | Tiempo |
|---|---|
| **Enterramiento minimo** | **24 horas** desde fallecimiento |
| **Enterramiento maximo** | **48 horas** desde fallecimiento |
| **Cadaver pasa a "restos cadavericos"** | A los **5 anios** |

### Tiempos de esterilizacion

| Metodo | Temperatura | Tiempo |
|---|---|---|
| **Autoclave programa 1** | 121oC | **15-20 minutos** |
| **Autoclave programa 2 (flash)** | 134oC | **3-7 minutos** |
| **Poupinel programa 1** | 170oC | **60 minutos (1 hora)** |
| **Poupinel programa 2** | 180oC | **30 minutos** |

### Tiempos legales y de contencion

| Concepto | Tiempo |
|---|---|
| **Contencion mecanica: revision** | Cada **15-30 minutos** |
| **Internamiento involuntario: comunicar al juz** | **24 horas** |
| **Internamiento involuntario: ratificacion judicial** | **72 horas** |

### Nemotecnicos:
- "Mascarilla = 2h, UPP = 2-4h, fallecido = 2h"
- "Rigor: 2-4h empieza, 12-15h maximo, 24-36h desaparece"
- "Enterrar: minimo 24h, maximo 48h"
- "Contencion: revisar cada 15-30 min"
- "Involuntario: 24h comunicar, 72h ratificar"

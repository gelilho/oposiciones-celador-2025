## Tests

La carpeta `tests/` contiene tests de examen reales y simulacros con las **respuestas correctas marcadas**.

Cada test tiene formato:
- Pregunta numerada
- 4 opciones (a, b, c, d)
- Respuesta correcta en **negrita**

## Como estudiar

1. **Lee cada tema** en orden, subrayando los datos clave (fechas, numeros, temperaturas, distancias)
2. **Repasa la seccion "DATOS CLAVE PARA EXAMEN"** al final de cada tema
3. **Haz los tests** sin mirar las respuestas, luego corrige
4. **Repite los tests fallados** hasta dominarlos
5. **Haz simulacros completos** con los examenes reales

## Datos imprescindibles que SIEMPRE caen en examen

- **23 funciones del celador** (art 14.2 Estatuto 1971): 22 especificas + 1 no especifica
- **Temperatura habitacion**: 20-22 C
- **Humedad**: 40-60%
- **Autoclave**: 121 C/15-20min o 134 C/3-7min
- **Triaje Manchester**: 5 niveles (rojo, naranja, amarillo, verde, azul)
- **Ambulancias**: A1, A2 (no asistenciales), B (SVB), C (SVA/UVI-movil)
- **Cama ortopedica/Judet**: la coloca el CELADOR
- **Camilla de palas**: solo movilizar, NUNCA trasladar
- **Internamiento involuntario**: art 763 LEC, 72h ratificacion judicial
- **Cadaver**: 5 primeros anos tras muerte (Decreto 151/2014)

## Legislacion clave

- Ley 41/2002: Autonomia del paciente

- Decreto 151/2014: Sanidad mortuoria de Galicia
- Ley 28/2005: Medidas sanitarias frente al tabaquismo
- RD Legislativo 1/2015: Medicamentos y productos sanitarios
